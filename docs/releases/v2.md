## Goal & Scope

v2.0 is the first major version bump and turns FeedFerret into a full audio reading experience. Podcast feeds live alongside RSS feeds in a unified list. Any article can be read aloud via Text-to-Speech. A native app wrapper (iOS + Android) brings push notifications via APNs/FCM and better offline support.

This is the largest release in the roadmap and must be split into independently mergeable sub-PRs. Each sub-PR must leave the app fully functional on its own.

## Out of Scope for v2.0

- Android/iOS system-level podcast sync (gpodder.net API)
- Transcript display and search (data model supports it; UI deferred)
- Lightning/Value4Value payments (data model stores `<podcast:value>`; no activation)
- Speech-to-text / voice search
- ElevenLabs or other TTS providers (can be added via BYOK mechanism without a major release)
- Podcast recommendation engine
